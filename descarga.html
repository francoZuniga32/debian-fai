bootear en windows

para poder grabar la imagen raw en un pendrive, tenemos que:

  1. Descargamos ImageWriter para Windows en este enlace: win32diskimager
  2. Arrancamos ImageWriter, seleccionamos la imagen ( archivo .image ) y seleccionamos la unidad donde la volcaremos. ¡OJO CON LA SELECCIÓN DE UNIDAD!
  3. Ya tenemos listo nuestro pendrive con la última compilación disponible, listo para ser enchufado en cualquier pc que nos permita arrancar desde usb.

luego de esto podremos reiniciar la PC. y continuar con la instalación

bootear en linux

Para grabar en el pendrive, utilizar el comando “dd”:

Primero se debe identificar el dispositivo USB con la aplicación “gparted” o el ejecutando el comando “fdisk -l” como administrador por terminal.

por ejemplo, si el pendrive fue reconocido como “/dev/sdb”, entonces el comando sería el siguiente:

$ dd if=/lugar/de/origen/debian-fai_v1-1.raw of=/dev/sdx bs=1M; sync

donde sdx es el pendrive

El proceso tarda varios minutos, se puede ver el progreso instalando previamente el paquete “pv”:

$ apt-get install pv
$ dd if=/lugar/de/origen/debian-fai_v1-1.raw |pv|dd of=/dev/sdx bs=1M; sync
