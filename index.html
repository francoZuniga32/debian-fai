¿Qué es Debian-fai?

Debian-fai es una distribución de linux basada en Debian Stretch (64bit) con el entorno de escritorio MATE, personalizada para la Facultad de Informática. Contiene gran parte de las aplicaciones necesarias para las materias de programación y también para uso general.
