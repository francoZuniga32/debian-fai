Instalacion BIOS

Configuración del setup

Antes de proceder con la instalación será necesario realizar algunas configuraciones en el setup. ingrese al setup y realice las siguientes operaciones:

  1. boot > boot mode: LEGACY
  2. boot > usb boot: Habilitado
  3. fast boot: Deshabilitado
  4. seguridad > secure boot: Deshabilitado ó seguridad > secure boot configuration > secure boot opción: desactivado

Debian-fai como sistema único

  1. Seleccionar la opción “Instalar Sistema”.
  2. Completar datos del usuario. Importante: establecer una contraseña de root.
  3. Identificar el disco donde se instalará el sistema, por ejemplo, en este caso “/dev/sda”:

  4. Crear las particiones necesarias para el sistema, como mínimo 2:

    - una partición para el área de intercambio “swap” a la cuál se recomienda darle el doble del tamaño de la ram.

    - una partición para el sistema raíz “/” que tendrá una dimensión mínima de 40 GB.

    Nota: se recomienda crear una partición extra para los datos del usuario “/home” que aloje los datos de usuario.

  5. Seleccionar el disco donde se instalará el gestor de arranque, en este caso “/dev/sda” y pulsar “siguiente”.
  6. Iniciar la instalación.
  7. Cuando la instalación finaliza se muestra el siguiente mensaje:
  8. Una vez hecho esto reiniciar el equipo y extraer el pendrive.

Debian-fai junto con otro sistema operativo

  1. Abrir la aplicación Gparted, ingresar como contraseña: “usuario”. Sistema -> Administración -> GParted
  2. Identificar una partición con espacio disponible. En este caso, utilizamos la partición “/dev/sdg2” que tiene disponibles 435GB. Dicha partición, tiene instalado el sistema operativo Windows 7, el cual deseamos conservar.
  3. Una vez identificada la partición objetivo, procedemos a redimensionarla para reducir el volumen y obtener así una nueva partición donde se podrá instalar el sistema.
  4. En este caso reducimos el volumen en 100GB, de manera que se puedan destinar 4GB como memoria de intercambio “swap” y los restantes 96GB para el sistema operativo debian-fai.
  5. Luego de redimensionar la partición, aparecerá un nuevo espacio “sin asignar”, deberá hacer click en la tilde verde para aplicar los cambios.
  6. Si el proceso fue exitoso se mostrará el siguiente mensaje:

  7. Para continuar, deberá cerrar la aplicación GParted, como indica la imagen anterior.

    En este momento, deberá iniciar con los pasos de la Opción 1, instalando debian-fai en la partición que hemos liberado con ese objetivo.
