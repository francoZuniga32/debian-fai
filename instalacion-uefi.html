Instalacion UEFI

Configuración del setup

Antes de proceder con la instalación será necesario realizar algunas configuraciones en el setup. Ingresar y realizar las siguientes operaciones:

  1. boot > boot mode: UEFI
  2. boot > usb boot: Habilitado
  3. fast boot: Deshabilitado
  4. guridad > secure boot: Deshabilitado ó seguridad > secure boot configuration > secure boot opción: desactivado

Nota: en ocasiones será necesario establecer una contraseña de administración para que se puedan realizar los cambios.

Debian-fai como sistema único

  1. Abrir la aplicación Gparted, ingresar como contraseña: “usuario”. Sistema -> Administración -> GParted
  2. Crear tabla de particiones nueva, seleccionar “gpt” y aceptar.
  3. Crear una partición fat32 de 100MB (Mínimo), con opciones “boot” y “esp”.
  4. También desde gparted se pueden crear las demas particiones: sistema raíz, swap, home (opcional). Caso contrario deberán crearse durante la instalación.

  5. Hacer doble click sobre “Instalar Sistema” e ingresar como contraseña: usuario

  6. Seleccionar la opción “Instalar Sistema”.

  7. Completar datos del usuario. Importante: establecer una contraseña de root.

  8. Identificar el disco donde se instalará el sistema, por ejemplo, en este caso “/dev/sda”:

  9. Crear las particiones necesarias para el sistema, como mínimo 2:

    - una partición para el área de intercambio “swap” a la cuál se recomienda darle el doble del tamaño de la ram.

    - una partición para el sistema raíz “/” que tendrá una dimensión mínima de 40 GB.

  10. Nota: se recomienda crear una partición extra para los datos del usuario “/home” que aloje los datos de usuario.

  11. Seleccionar el disco donde se instalará el gestor de arranque, en este caso “/dev/sda” y pulsar “siguiente”.

  12. Iniciar la instalación.

  13. Cuando la instalación finaliza se muestra el siguiente mensaje:

  14. A partir de aquí el sistema estará instalado, solo resta configurar el grub para un correcto arranque, los pasos para lograrlo se describen en la sección instalación de grub uefi más adelante.
  15. Una vez hecho esto reiniciar el equipo y extraer el pendrive.

Debian-fai junto con otro sistema operativo

  1. Abrir la aplicación Gparted, ingresar como contraseña: “usuario”. Sistema -> Administración -> GParted
  2. Procedemos a redimensionar la partición donde tenemos instalado el sistema operativo que deseamos mantener para reducir el volumen y obtener así una nueva partición donde se podrá instalar el sistema(mínimo 50 gb).

  3. Hacer doble click sobre “Instalar Sistema” e ingresar como contraseña: “usuario”.

  4. Seleccionar la opción “Instalar Sistema”.

  5. Completar datos del usuario.

    Importante: establecer una contraseña de root.

  6. Identificar el disco donde se instalará el sistema, por ejemplo, en este caso “/dev/sda”:

  7. En la particion que hemos creado al principio, crear las particiones necesarias para el sistema, como mínimo 2:

    - una partición para el área de intercambio “swap” a la cuál se recomienda darle el doble del tamaño de la ram.

    - una partición para el sistema raíz “/” que tendrá una dimensión mínima de 40 GB.

    Nota: se recomienda crear una partición extra para los datos del usuario “/home” que aloje los datos de usuario, lo cuál implica reducir la partición del SO que se desea conservar en más de 50 gb.

  8. Iniciar la instalación.

  9. Cuando la instalación finaliza se muestra el siguiente mensaje:

  10. A partir de aquí el sistema estará instalado, solo resta configurar el grub para un correcto arranque, los pasos para lograrlo se describen en la sección instalación de grub uefi más adelante.
  11. Una vez hecho esto reiniciar el equipo y extraer el pendrive.
